MyBusiness - Spring Boot E-Commerce ApplicationThis is a full-stack Java application built using Spring Boot, Spring Data JPA, and Thymeleaf. It manages Admins, Users, Products, and Orders with a complete CRUD implementation and business logic for price calculations.## Project StructureThe project follows the standard Maven Layered Architecture:entities: JPA models mapped to database tables (Admin, User, Product, Orders).repositories: Interfaces extending CrudRepository or JpaRepository for DB operations.services: Business logic layer (authentication, calculations, and data processing).controllers: Web layer handling HTTP requests and navigation.loginCredentials: DTO (Data Transfer Objects) for handling login form data.basiclogics: Utility classes for specific calculations (e.g., Logic.java).## API Documentation### 1.
